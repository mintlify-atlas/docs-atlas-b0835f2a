## Overview

YouTube Manager uses two primary Django models to manage video data: `VideoGuardado` for saved favorite videos and `VideoSubido` for videos uploaded through the CMS.

## VideoGuardado

Stores videos that users have saved as favorites from YouTube search results.

### Model Definition

```python
from django.db import models
from django.contrib.auth.models import User

class VideoGuardado(models.Model):
    user = models.ForeignKey(User, on_delete=models.CASCADE)
    video_id = models.CharField(max_length=50)
    titulo = models.CharField(max_length=255)
    canal = models.CharField(max_length=255)
    descripcion = models.TextField()
    miniatura_url = models.URLField()
    fecha_guardado = models.DateTimeField(auto_now_add=True)

    class Meta:
        unique_together = ('user', 'video_id')

    def __str__(self):
        return self.titulo
```

### Fields

<ParamField path="user" type="ForeignKey">
  Reference to Django's User model. Cascade deletion - when a user is deleted, all their saved videos are removed.
</ParamField>

<ParamField path="video_id" type="CharField">
  YouTube video identifier (max 50 characters). This is the unique ID from YouTube's API.
</ParamField>

<ParamField path="titulo" type="CharField">
  Video title (max 255 characters).
</ParamField>

<ParamField path="canal" type="CharField">
  Channel name that uploaded the video (max 255 characters).
</ParamField>

<ParamField path="descripcion" type="TextField">
  Video description. Unlimited length text field.
</ParamField>

<ParamField path="miniatura_url" type="URLField">
  URL to the video's thumbnail image from YouTube.
</ParamField>

<ParamField path="fecha_guardado" type="DateTimeField">
  Timestamp when the video was saved. Automatically set on creation (`auto_now_add=True`).
</ParamField>

### Meta Options

<ParamField path="unique_together" type="tuple">
  Constraint: `('user', 'video_id')` - Ensures each user can only save a specific video once.
</ParamField>

### Methods

<ResponseField name="__str__()" type="string">
  Returns the video title for string representation.
</ResponseField>

### Database Schema

| Column | Type | Constraints |
|--------|------|-------------|
| id | Integer | PRIMARY KEY, AUTO_INCREMENT |
| user_id | Integer | FOREIGN KEY (auth_user.id), NOT NULL |
| video_id | VARCHAR(50) | NOT NULL |
| titulo | VARCHAR(255) | NOT NULL |
| canal | VARCHAR(255) | NOT NULL |
| descripcion | TEXT | NOT NULL |
| miniatura_url | VARCHAR(200) | NOT NULL |
| fecha_guardado | DATETIME | NOT NULL |

<Note>
  The `unique_together` constraint prevents duplicate entries: a user cannot save the same YouTube video multiple times.
</Note>

## VideoSubido

Tracks videos that users have uploaded to YouTube through the application's CMS functionality.

### Model Definition

```python
class VideoSubido(models.Model):
    ESTADOS = [
        ('private', 'Privado'),
        ('public', 'Público'),
        ('unlisted', 'No listado'),
    ]

    user = models.ForeignKey(User, on_delete=models.CASCADE)
    video_id = models.CharField(max_length=50, unique=True)
    titulo = models.CharField(max_length=255)
    descripcion = models.TextField()
    estado = models.CharField(max_length=20, choices=ESTADOS, default='private')
    fecha_subida = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return f"{self.titulo} ({self.estado})"
```

### Fields

<ParamField path="user" type="ForeignKey">
  Reference to Django's User model. Links uploaded videos to the user who uploaded them. Cascade deletion enabled.
</ParamField>

<ParamField path="video_id" type="CharField">
  YouTube video identifier (max 50 characters). Must be globally unique across all users (`unique=True`).
</ParamField>

<ParamField path="titulo" type="CharField">
  Video title (max 255 characters).
</ParamField>

<ParamField path="descripcion" type="TextField">
  Video description. Unlimited length text field.
</ParamField>

<ParamField path="estado" type="CharField">
  Privacy status of the video. Max 20 characters. Must be one of the `ESTADOS` choices. Defaults to `'private'`.
</ParamField>

<ParamField path="fecha_subida" type="DateTimeField">
  Timestamp when the video was uploaded. Automatically set on creation (`auto_now_add=True`).
</ParamField>

### Privacy Status Choices

The `ESTADOS` constant defines valid privacy states:

| Value | Display Name | Description |
|-------|-------------|-------------|
| `private` | Privado | Only the uploader can view the video |
| `public` | Público | Anyone can find and view the video |
| `unlisted` | No listado | Only people with the link can view |

### Methods

<ResponseField name="__str__()" type="string">
  Returns a formatted string: `"{titulo} ({estado})"`, e.g., "My Video (private)".
</ResponseField>

### Database Schema

| Column | Type | Constraints |
|--------|------|-------------|
| id | Integer | PRIMARY KEY, AUTO_INCREMENT |
| user_id | Integer | FOREIGN KEY (auth_user.id), NOT NULL |
| video_id | VARCHAR(50) | UNIQUE, NOT NULL |
| titulo | VARCHAR(255) | NOT NULL |
| descripcion | TEXT | NOT NULL |
| estado | VARCHAR(20) | NOT NULL, DEFAULT 'private' |
| fecha_subida | DATETIME | NOT NULL |

<Note>
  Unlike `VideoGuardado`, the `video_id` field has a global `unique=True` constraint because YouTube video IDs are globally unique.
</Note>

## Relationships

Both models have a **many-to-one** relationship with Django's `User` model:

- One user can have many saved videos (`VideoGuardado`)
- One user can upload many videos (`VideoSubido`)
- Deleting a user cascades to delete all related videos

### Accessing Related Videos

```python
# Get all saved videos for a user
user.videoguardado_set.all()

# Get all uploaded videos for a user
user.videosubido_set.all()

# Count saved videos
user.videoguardado_set.count()
```

## Usage Examples

### Creating a Saved Video

```python
video = VideoGuardado.objects.create(
    user=request.user,
    video_id='dQw4w9WgXcQ',
    titulo='Example Video',
    canal='Example Channel',
    descripcion='This is a sample video',
    miniatura_url='https://i.ytimg.com/vi/dQw4w9WgXcQ/hqdefault.jpg'
)
```

### Using get_or_create for Saved Videos

```python
video, created = VideoGuardado.objects.get_or_create(
    user=request.user,
    video_id=video_id,
    defaults={
        'titulo': titulo,
        'canal': canal,
        'descripcion': descripcion,
        'miniatura_url': miniatura
    }
)
```

### Creating an Uploaded Video Record

```python
VideoSubido.objects.create(
    user=request.user,
    video_id=response['id'],
    titulo='My Uploaded Video',
    descripcion='Description here',
    estado='private'
)
```

### Querying Videos

```python
# Get user's saved videos, newest first
saved = VideoGuardado.objects.filter(user=request.user).order_by('-fecha_guardado')

# Get user's uploaded videos, newest first
uploaded = VideoSubido.objects.filter(user=request.user).order_by('-fecha_subida')

# Get public uploaded videos
public_videos = VideoSubido.objects.filter(estado='public')
```
