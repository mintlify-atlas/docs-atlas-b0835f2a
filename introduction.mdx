## Welcome to YouTube Manager

YouTube Manager is a powerful Django web application that integrates with the YouTube Data API to provide a comprehensive video management platform. Upload videos, search YouTube's catalog, save favorites, and manage privacy settings—all from a clean, user-friendly interface.

## Technology Stack

YouTube Manager is built with modern, reliable technologies:

- **Django 6.0** - Python web framework for rapid development
- **MySQL** - Robust relational database for data persistence
- **Google API Client** - Official Python client for YouTube Data API v3
- **OAuth 2.0** - Industry-standard authentication protocol
- **Python 3.x** - Modern Python runtime environment
